# GoldView

This project is a complete website for GoldView, featuring a React frontend and a Python backend.

## Project Structure

- `backend/`
  - Contains the Python backend code.
  - `app.py`: The main application file for the backend.
  - `requirements.txt`: The list of Python dependencies.

- `frontend/`
  - Contains the React frontend code.
  - `package.json`: The Node.js dependencies and scripts.

- `components/`
  - `Dashboard.js`: React component for the dashboard.
  - `Calculator.js`: React component for the calculator.
  - `PriceCharts.js`: React component for price charts.

- `services/`
  - `api.js`: Contains functions for API services to interact with the backend.

## Getting Started

### Backend
1. Navigate to the `backend` directory.
2. Install the required packages using `pip install -r requirements.txt`.
3. Run the application with `python app.py`.

### Frontend
1. Navigate to the `frontend` directory.
2. Install the required packages using `npm install`.
3. Start the application with `npm start`.
