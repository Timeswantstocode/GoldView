# Palette's Journal - GoldView

## 2025-05-14 - Accessibility and Interactive Feedback
**Learning:** Many interactive elements in the dashboard were missing proper ARIA labels and keyboard focus indicators, making them difficult to use for screen reader users and keyboard navigators.
**Action:** Always add `aria-label` to icon-only buttons, proper `role` and `aria-checked` to toggles, and ensure focus visible states are present for all interactive elements.
